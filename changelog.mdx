<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time order tracking dashboard with live updates for kitchen and pickup counters.
- Introduced loyalty program API endpoints: `POST /api/loyalty/points` and `GET /api/loyalty/customers/{id}`.
- New inventory alerts send webhooks to `https://your-webhook-url.com/webhook` when stock falls below thresholds.

## Improvements

- Optimized menu item search to reduce query time by 40%.
- Enhanced mobile responsiveness for tablet-based point-of-sale terminals.

## Bug Fixes

- Fixed order duplicate submission issue during peak hours.
- Resolved PDF receipt generation errors for special characters in customer names.

</Update>
