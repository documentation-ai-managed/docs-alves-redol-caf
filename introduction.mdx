## Overview

Alves Redol Café provides a centralized hub for your project documentation. You create, organize, and collaborate on docs with ease. Use it to build knowledge bases, API references, and user guides that scale with your team.

<Callout kind="info">
  Alves Redol Café uses brand color `#3B82F6` for primary elements. Apply it consistently in custom themes and embeds.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.alvesredol.com/register` with your email.
  </Step>
  <Step title="New Space" icon="plus">
    Click "New Space" and name it for your project, like "API Docs".
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Create a page titled "Introduction". Use the editor to add content.
  </Step>
  <Step title="Publish" icon="globe">
    Hit "Publish" to make your docs live. Share the URL with your team.
  </Step>
</Steps>

## Brand Guidelines

Follow these to maintain consistency.

Use the primary brand color `#3B82F6` for buttons, links, and accents. Secondary colors include shades of gray for text.

```css
:root {
  --primary: #3B82F6;
  --primary-hover: #2563EB;
}
```

<Expandable title="Custom Theme Example" default-open="false">
  Apply your theme via settings.

  <CodeGroup tabs="CSS,JSON">
  ```css
  .doc-button {
    background-color: var(--primary);
    border-radius: 6px;
    padding: 8px 16px;
  }
  ```
  ```json
  {
    "theme": {
      "primaryColor": "#3B82F6"
    }
  }
  ```
  </CodeGroup>
</Expandable>
